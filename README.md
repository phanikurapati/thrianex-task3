# To-Do List Application

A fully interactive client-side To-Do List application built using HTML, CSS, and JavaScript. This project demonstrates DOM manipulation, event handling, state management, CRUD operations, and browser-based data persistence using Local Storage.

## Features

### CRUD Functionality

* Create new tasks
* View existing tasks
* Edit task names
* Delete tasks

### Task Management

* Mark tasks as completed
* Undo completed tasks
* Real-time UI updates

### Advanced Filtering

* View all tasks
* View active tasks only
* View completed tasks only

### Data Persistence

* Automatically saves tasks using `window.localStorage`
* Tasks remain available after page refresh or browser restart

### Modern JavaScript Concepts

* Dynamic DOM manipulation
* Event delegation
* State-driven rendering
* ES6+ syntax
* Array methods (`map`, `filter`, `find`)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-app.git
```

2. Navigate to the project folder:

```bash
cd todo-app
```

3. Open `index.html` in your browser.

No additional dependencies or installation steps are required.

## Usage

### Add a Task

Type a task into the input field and click **Add**.

### Complete a Task

Click the **Done** button to mark a task as completed.

### Edit a Task

Click the **Edit** button and enter the updated task text.

### Delete a Task

Click the **Delete** button to permanently remove a task.

### Filter Tasks

Use the filter buttons to display:

* All Tasks
* Active Tasks
* Completed Tasks
